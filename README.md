# Calculadora usando RUST
Esto es una calculadora que parsea la entrada de un usuario desde stdin y en stdout escribe el resultado de calculado.

![imagen](https://pbs.twimg.com/media/E63X37_WUAEOXPg.jpg)


## Como usar

```bash
cargo run
```
---

```bash
Calcular: 2 + 2 * 4
```

```bash
Resultado: 10
```
